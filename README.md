# Turbo-Racing-Multiplayer
A 2-player Racing Game Made with Native Javascript

## Controls
### Player 1(Red Car):
#### W - Accelerate
#### A - Turn Left
#### S - Brake/Reverse
#### D - Turn Right

### Player 2(Blue Car):
#### ⬆️ - Accelerate
#### ⬅️ - Turn Left
#### ⬇️ - Brake/Reverse
#### ➡️ - Turn Right
